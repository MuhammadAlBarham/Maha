############
Contributing
############

Thank you for your interest in contributing to Maha! Small improvements or bug fixes are
always appreciated. In addition to writing code, Maha accepts many kinds of contributions, including:

*  Help with outreach and onboard new contributors
*  Maintain and improve our documentation
*  Add more edge cases to our test suite
*  Review and approve `pull requests <https://github.com/TRoboto/Maha/pulls>`_
*  Report bugs and provide feedback `directly on GitHub <https://github.com/TRoboto/Maha/issues>`_
*  Develop tutorials, presentations, and other educational materials

.. note::
    However you decide to contribute or interact with the community,
    please always be civil and respect other members of the community.
